**1920×1080px** (16:9) | 安全边距: 96px

## Global Rules (全局基础规范)

### 1.1 画布尺寸
| 画布类型 | 标准尺寸 | 适用场景 |
|----------|----------|----------|
| 高清主画布（推荐） | 1920px × 1080px | 99%商业演示、投影、线上会议 |
| 标清兼容画布 | 1333px × 750px | 老旧投影设备、低分辨率适配 |
| 打印适配画布 | 2540px × 1430px（300DPI） | 需打印成册的演示文稿 |

### 1.2 安全边距
- 左右边距：≥96px（画布左右各预留5%安全区）
- 上下边距：≥80px（画布上下各预留7%安全区）
- 栅格系统：采用8pt增量栅格，所有尺寸必须为8的整数倍
- 最小间距：图片与图片、图片与文字的间距≥24px

### 1.3 核心设计原则
1. **对比Contrast**：主图与副图尺寸差≥30%，明确视觉层级
2. **重复Repetition**：同系列PPT的图片比例、圆角、描边风格完全统一
3. **对齐Alignment**：所有图片必须与栅格线对齐，禁止无规则错位
4. **亲密性Proximity**：相关联的图片间距≤40px，非关联内容间距≥80px

### 1.4 图片比例强制规范
| 标准比例 | 适用场景 | 禁止场景 |
|----------|----------|----------|
| 3:4（竖版） | 单品展示、人物肖像、垂直场景图 | 禁止压缩为细长条（宽高比＞1:3） |
| 4:3（横版） | 场景图、穿搭图、横向产品展示 | 禁止拉伸为扁长条（宽高比＞3:1） |
| 16:9（全屏） | 封面页、章节过渡页、全场景氛围图 | 禁止用于内容页多图布局 |
| 1:1（方形） | 图标、logo、九宫格多品展示 | 禁止用于主视觉单品展示 |

> **比例偏差不得超过3%**，所有图片必须等比缩放，禁止强制拉伸变形

## <a name="single"></a>Single Image Layout

### 单大图 (左图右文/左文右图)
```
┌────────────────┬──────────────┐
│   文字 40%     │  图片 55%    │
│                │  4:3横版     │
└────────────────┴──────────────┘
```
- 图片宽度：1000px，高度：750px
- 图片X坐标：824px，Y坐标：165px

```xml
<rect x="824" y="165" width="1000" height="750" rx="8" fill="#252530" stroke="#3a3a45"/>
<text x="1324" y="530" text-anchor="middle" fill="#6e6e73">[ 主图 4:3 1000×750 ]</text>
```

### 单大图居中 (封面/章节页)
```
┌─────────────────────────────────────────────┐
│                                             │
│               [ 全屏主视觉图 ]              │
│               16:9 比例                     │
│                                             │
└─────────────────────────────────────────────┘
```
- 16:9全屏图：宽度1728px，高度972px，X=96px，Y=54px

```xml
<rect x="96" y="54" width="1728" height="972" rx="8" fill="#252530" stroke="#3a3a45"/>
<text x="960" y="540" text-anchor="middle" fill="#6e6e73">[ 16:9全屏 1728×972 ]</text>
```

**适用场景**：单产品讲解、趋势定义、单风格解析、核心数据配图、培训内容页、封面页、章节过渡页

## <a name="dual"></a>Dual Image Layout

### 双图横排 (对比布局)
```
┌──────────────┬──────────────┐
│   左图 4:3   │   右图 4:3   │
│   832×624   │   832×624   │
└──────────────┴──────────────┘
```
- 单图宽度：832px，高度：624px（4:3）
- 左图X=96px，右图X=992px，Y统一=165px
- 双图间距：64px

```xml
<rect x="96" y="165" width="832" height="624" rx="8" fill="#252530" stroke="#3a3a45"/>
<text x="512" y="477" text-anchor="middle" fill="#6e6e73">[ 左图 4:3 ]</text>

<rect x="992" y="165" width="832" height="624" rx="8" fill="#252530" stroke="#3a3a45"/>
<text x="1408" y="477" text-anchor="middle" fill="#6e6e73">[ 右图 4:3 ]</text>
```

**适用场景**：风格对比、产品对比、前后效果对比、国内vs海外市场对比、竞品分析

### 双图竖排 (时间线/流程)
```
┌─────────────────────────┐
│      上图 4:3           │
│      1040×390           │
├─────────────────────────┤
│      下图 4:3           │
│      1040×390           │
└─────────────────────────┘
```
- 宽度：1040px，单图高度：390px
- 上图Y=100，下图Y=530

```xml
<rect x="96" y="100" width="1040" height="390" rx="8" fill="#252530" stroke="#3a3a45"/>
<text x="616" y="295" text-anchor="middle" fill="#6e6e73">[ 上图 4:3 ]</text>

<rect x="96" y="530" width="1040" height="390" rx="8" fill="#252530" stroke="#3a3a45"/>
<text x="616" y="725" text-anchor="middle" fill="#6e6e73">[ 下图 4:3 ]</text>
```

**适用场景**：时间线、流程步骤、阶段展示

## <a name="multi"></a>Multi-Image Layout

### 四宫格 (多产品展示)
```
┌─────────┬─────────┐
│  左上   │  右上   │
│  832×416│  832×416│
├─────────┼─────────┤
│  左下   │  右下   │
│  832×416│  832×416│
└─────────┴─────────┘
```
- 单图宽度：832px，高度：416px
- 横向间距：64px，纵向间距：40px
- 坐标：左上X=96/Y=120，右上X=992/Y=120，左下X=96/Y=576，右下X=992/Y=576

```xml
<rect x="96" y="120" width="832" height="416" rx="8" fill="#252530" stroke="#3a3a45"/>
<rect x="992" y="120" width="832" height="416" rx="8" fill="#252530" stroke="#3a3a45"/>
<rect x="96" y="576" width="832" height="416" rx="8" fill="#252530" stroke="#3a3a45"/>
<rect x="992" y="576" width="832" height="416" rx="8" fill="#252530" stroke="#3a3a45"/>
```

**适用场景**：多单品展示、多风格合集、穿搭look合集、细节展示、多场景配图

### 三图横排
```
┌──────────────┬──────────────┬──────────────┐
│   左图      │   中图       │   右图       │
│  544×408    │  544×408     │  544×408     │
└──────────────┴──────────────┴──────────────┘
```
- 单图宽度：544px，高度：408px
- X坐标分别为96px、688px、1280px，Y=165px
- 图片间距：48px

```xml
<rect x="96" y="165" width="544" height="408" rx="8" fill="#252530" stroke="#3a3a45"/>
<rect x="688" y="165" width="544" height="408" rx="8" fill="#252530" stroke="#3a3a45"/>
<rect x="1280" y="165" width="544" height="408" rx="8" fill="#252530" stroke="#3a3a45"/>
```

**适用场景**：三风格对比、三阶段时间线、三市场数据、多品并列展示、流程步骤配图

## <a name="main-sub"></a>Main + Sub Layout

### 主图 + 双副图
```
┌──────────────┬─────────┐
│              │  右上    │
│    主图      │  544×400│
│   1120×840  ├─────────┤
│              │  右下    │
│              │  544×400│
└──────────────┴─────────┘
```
- 主图宽度：1120px，高度：840px，X=96px，Y=120px
- 副图宽度：544px，高度：400px，X=1280px
- 上副图Y=120px，下副图Y=560px
- 主副图间距：64px

```xml
<rect x="96" y="120" width="1120" height="840" rx="8" fill="#252530" stroke="#3a3a45"/>
<rect x="1280" y="120" width="544" height="400" rx="8" fill="#252530" stroke="#3a3a45"/>
<rect x="1280" y="560" width="544" height="400" rx="8" fill="#252530" stroke="#3a3a45"/>
```

**适用场景**：核心单品+细节展示、主风格+细分风格展示、主场景+辅助场景、核心数据+佐证配图

## Content Management Rules

### 单页内容核心规则
1. **一页一主题**：每个页面仅聚焦1个核心主题，禁止多主题强行塞入同一页面
2. **图文比例管控**：
   - 内容页：文字区≤页面40%，图片区≥页面50%，留白≥10%
   - 封面/过渡页：图片区≥页面80%，文字区≤页面20%
3. **拆分规则**：当内容+图片占用超过页面85%空间时，必须拆分为2个及以上页面，禁止缩小字号、压缩间距强行塞入
4. **字号规范**：标题≥28pt，正文≥16pt，注释≥12pt

### 图片内容规范
1. 所有图片必须清晰度高，禁止使用模糊、低清、有水印的图片
2. 图片必须与页面主题强相关，禁止使用无意义的装饰图
3. 人物/产品图片必须保持完整，禁止裁切关键部位

## Decision Tree (布局选型决策树)

```
需要几张图?
├── 1张
│   ├── 封面/章节页 → 全屏布局
│   └── 内容页 → 单大图(左图右文/左文右图)
│
├── 2张
│   ├── 对比 → 双图横排
│   └── 时间线/流程 → 双图竖排
│
├── 3张 → 三图横排
│
└── 4张+
    ├── 多产品展示 → 四宫格
    └── 内容过多 → 拆分多页
```

## Layout Selection Guide (布局选型表)

| 页面内容类型 | 首选布局 | 次选布局 | 禁止布局 |
|--------------|----------|----------|----------|
| 封面/章节过渡页 | 全屏布局 | 左文右图 | 多图布局 |
| 单主题核心内容 | 单大图左右布局 | 单大图居中 | 三图及以上 |
| 2项内容对比 | 双图横排 | 双图竖排 | 四宫格 |
| 3项并列内容 | 三图横排 | 主图+双副图 | 四宫格 |
| 4项及以上 | 四宫格 | 拆分多页 | 单张大图 |
| 核心单品+细节 | 主图+双副图 | 左文右图+小图 | 等宽多图 |

## Forbidden List (禁忌清单)

1. **禁止拉伸图片**改变原始宽高比，强制变形
2. **禁止细长条/扁长条**图片，必须保持3:4或4:3比例
3. **禁止同页面内**同类型图片尺寸、比例、圆角不统一
4. **禁止图片超出**安全边距，导致播放时被裁切
5. **禁止图片与文字**间距小于24px，导致视觉拥挤
6. **禁止单页面内**超过4张以上无层级的图片
7. **禁止使用**低清、水印、模糊的图片作为正式内容
8. **禁止在复杂图片**上叠加无背景的浅色文字
